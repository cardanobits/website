TECHIE STUFF

A Non-Fungible Token (NFT) means the item is unique, so how you create them? By using a signature and time-locked minting policy, so once the time has come, no more tokens can be minted using that policy.

How can I know how many tokens are minted under the same policy? Well, you can check the policy using: cardanoscan.io/tokenPolicy/POLICYcardanoscan.io/tokenPolicy/POLICY

This NFT has metadata? Yes, it's stored on the Cardano blockchain. Every property or detail of each NFT is stored on the blockchain, with a hash to link its file stored with IPFS, using the CID to keep each file immutable and retrievable forever.

How can I see the image of my NFT? Good one, you must grab the code (CID) that you see in the "file" field, of the metadata of your NFT and put it in your browser like this: ipfs.io/ipfs/CID
